# Ambient Audio Files

Place MP3 loops here to replace the synthesized ambient soundscapes.
When a file is present, it takes priority over the procedural fallback.

## Expected filenames

| File | Story theme | Mood |
|------|-------------|------|
| `nature.mp3` | Forests, animals, outdoors | Bright, open |
| `ocean.mp3` | Sea, water, sailing | Deep, flowing |
| `space.mp3` | Sci-fi, stars, cosmos | Ethereal, vast |
| `peaceful.mp3` | Calm stories, bedtime | Gentle, soothing |
| `mysterious.mp3` | Mysteries, magic, caves | Dark, moody |
| `dramatic.mp3` | Action, conflict, storms | Tense, suspenseful |
| `adventure.mp3` | Quests, journeys, heroes | Energetic, upbeat |
| `celebration.mp3` | Victories, parties, joy | Joyful, bright |

## Requirements

- **Format:** MP3 (128–192 kbps recommended)
- **Duration:** 30–90 seconds, seamlessly loopable
- **Volume:** Normalize to -18 to -14 LUFS (these play as background ambience)
- **License:** CC0 / Public Domain only

## Recommended sources (all CC0 / royalty-free)

- [Pixabay Music](https://pixabay.com/music/) — search "ambient loop"
- [Freesound.org](https://freesound.org/) — filter by CC0 license
- [OpenGameArt.org](https://opengameart.org/) — search "ambient CC0"
- [Free Music Archive](https://freemusicarchive.org/genre/Ambient) — check license per track
